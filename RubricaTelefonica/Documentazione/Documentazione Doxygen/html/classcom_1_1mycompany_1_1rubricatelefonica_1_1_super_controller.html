Detailed Description

Gestisce centralmente una lista di contatti visibili in una tabella.

Questa classe contiene una lista di contatti (ObservableList<Contatto>) che può essere condivisa tra altre classi per visualizzare, aggiungere o modificare contatti nella rubrica. L'invariante della classe è che lista non può mai essere null, e deve contenere oggetti di tipo Contatto.
