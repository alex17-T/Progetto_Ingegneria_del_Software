Detailed Description

Questa classe gestisce l'interfaccia principale della rubrica telefonica.

Questo file contiene la logica dell'interfaccia principale. Questo file implementa i metodi che rendono interattiva e correttamente funzionante l'interfaccia grafica codificata nel file Home.fxml. FXML Controller class

◆ AggiungiContatto()

void com.mycompany.rubricatelefonica.HomeController.AggiungiContatto ( ActionEvent event) throws IOException

Il metodo apre una finestra per l’inserimento dei dati del nuovo contatto

Questo metodo viene invocato al clic sul pulsante "Aggiungi". Viene aperta una nuova finestra per l'inserimento nella tabella dei dati: nome, cognome, numeri di telefono ed indirizzi email. Dopo l'inserimento è possibile vedere il contatto in tabella

Parameters
eventl'evento è generato dall'utente tramite interazione con l'interfaccia grafica quando desidera aggiungere un contatto.
Exceptions
IOExceptionse il contatto che l'utente cerca di aggiungere è già presente in rubrica

◆ initialize()

void com.mycompany.rubricatelefonica.HomeController.initialize ( URL url,
ResourceBundle rb )

Il metodo configura la tabella e le sue colonne.

Questo metodo inizializza le colonne della tabella riferendole agli attributi nome, cognome, numTel1 e email1 di Contatto. La tabella viene anche settata come "senza elementi selezionati" e gli viene associato un ChangeListener per attivarsi se il contatto selezionato cambia. Rende la lista interattiva filtrandola e ordinadola

Precondition
La lista dei contatti (cioè SuperController.lista) è inizializzata.
Postcondition
La tabella contiene i contatti della lista osservabile.
